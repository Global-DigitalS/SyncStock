**Key Technical Concepts**
- **Integración con Stripe Checkout:** Creación de sesiones de pago y redirección del usuario para procesar suscripciones.
- **Tareas en Segundo Plano (Background Tasks):** Uso de  para ejecutar operaciones largas (como la sincronización) sin bloquear la respuesta de la API.
- **Sondeo de API (Polling):** El frontend consulta periódicamente un endpoint de estado para mostrar el progreso en tiempo real de una tarea en segundo plano.
- **Optimización de API de Terceros:** Uso de  para reutilización de conexiones (Dolibarr) y operaciones por lotes (WooCommerce) para reducir drásticamente el número de llamadas de red.
- **Contenido Dinámico (Headless CMS):** La landing page obtiene su contenido de la base de datos a través de una API, permitiendo su edición desde un panel de administración sin necesidad de redesplegar el código.
